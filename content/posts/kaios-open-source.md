# KaiOS Open Source Resources

Here is a compilation of [KaiOS](https://www.kaiostech.com/) apps, tools, and resources to find inspiration and help you get started on your next endeavor. This list is by no means exhaustive, but attempts to cover a wide range of available open source projects related to KaiOS.

* [gecko-b2g](https://github.com/kaiostech/gecko-b2g) - KaiOS' MPL-licensed fork of Boot 2 Gecko (B2G), previously known as Firefox OS
* [api-daemon](https://github.com/kaiostech/api-daemon) - Rust-based external API for KaiOS, esp. 3.0+

## User Interface

* [KaiUI](https://github.com/AdrianMachado/KaiUI) - React-based UI library (last [code commit](https://github.com/AdrianMachado/KaiUI/commit/bf764f2d7446a3d45b731075d57d0d30cbb0e255) in April 2021)
* [Vue KaiUI](https://github.com/sebastianbaar/vue-kaiui) - KaiUI written for Vue.js
* [Arrow key navigation](https://github.com/nolanlawson/arrow-key-navigation)
* [KaiUIng](https://github.com/farooqkz/KaiUIng) - CSS library
* [Onyx UI](https://github.com/garredow/onyx-ui) - Svelte component library
* [Mai UI](https://github.com/garredow/mai-ui) - Preact component library

## Tools

* [appscmd](https://github.com/kaiostech/appscmd) - utility for installs KaiOS apps
* [KaiOS Web Suite](https://github.com/arma7x/kaios-web-suite) - PC suite for managing KaiOS 2.5 devices
* [KaiWare](https://github.com/garredow/kaiware) - Desktop app for managing KaiOS devices, written in TypeScript
* [KaiOS Lib](https://github.com/garredow/kaios-lib) - Library for interacting with KaiOS APIs
* [SQLite Scraper](https://github.com/joedinsmoor/KaiOS.Scraper)
* [Wallace Toolbox](https://gitlab.com/suborg/wallace-toolbox) - Universal toolbox for internal tweaking and rooting, based on [Libwallace](https://gist.github.com/plugnburn/00fa61006513cdb0a12adf61a6e425e1)
* [OrigAMI](https://gitlab.com/suborg/origami) - Origins of Application Marketless Installation
* [CrossTweak](https://gitlab.com/suborg/crosstweak) - Lightweight KaiOS tweaker with 10 functions
* [Project Kava](https://gitlab.com/suborg/project-kava) - Java 2 Mobile Edition (J2ME) emulator for KaiOS
